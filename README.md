# YouTube Trend Tracer

A full-stack app that fetches trending YouTube videos from multiple regions, analyzes engagement, and displays them in a filterable web UI with country, language, category, and time filters.

## Features

- **Multi-region trends** — Fetches from 10 regions (India, US, UK, Canada, Australia, Germany, France, Japan, Brazil, South Korea)
- **Language detection** — Detects title language for filtering (e.g. English, Hindi)
- **Filters** — Country, language, category, viral, engagement level, and date range
- **Viral classifier** — Random Forest model labels top 25% by views as “viral”
- **Deploy-ready** — Frontend on GitHub Pages; backend on Render

## Tech Stack

| Part     | Stack |
|----------|--------|
| Backend  | Python, Flask, pandas, scikit-learn, YouTube Data API |
| Frontend | HTML, CSS, JavaScript (vanilla) |
| Deploy   | GitHub Pages (frontend), Render (API) |

## Quick Start

### 1. Clone and install

```bash
git clone https://github.com/AyushVerma24/youtube_trend_teacer.git
cd youtube_trend_teacer
```

Install Python dependencies (project root):

```bash
pip install google-api-python-client pandas numpy matplotlib seaborn scikit-learn textblob wordcloud nltk flask flask-cors langdetect
```

### 2. Generate trend data (optional)

```bash
python main.py
```

This fetches trending videos from all regions, detects language, runs the viral classifier, and saves `youtube_api_cleaned_data.csv`. You can also use **Refresh trends** in the UI after the backend is running.

### 3. Run locally

```bash
cd backend
python app.py
```

Open **http://localhost:5000**. The backend serves both the API and the frontend.

## API

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/api/trends` | GET | Returns trends from CSV (`trends`, `count`) |
| `/api/trends/refresh` | POST | Re-runs `main.py` and returns updated trends (can take ~1–2 min) |

## Deployment
